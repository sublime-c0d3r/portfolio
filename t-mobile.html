T-Mobile Deals Hub

T-Mobile is the brand name used by the mobile communications subsidiaries of the German telecommunications company Deutsche Telekom AG. The brand is also active in the Czech Republic (T-Mobile Czech Republic), the Netherlands (T-Mobile Netherlands), Poland (T-Mobile Polska), and the United States (T-Mobile US).

Project Overview

T-Mobile deals hub is a web application which is written in React/Redux. This is a responsive web application targeted for both coumpter and mobile users. T-Mobile customers can login to their T-Mobile account to order various electronice gadgets, apply promo codes for discounts, request for refunds, compare gadgets and contact customer care personnels.

This application uses Adobe AEM for content management. I managed a team of 8 developers who workd seamlessly with each other for making life changing enhancements/fixes and for delivering changes on a timely manner. Some of the important milestones include; removal of unneccessary libraires and some code optimizations to achieve 60% performance boost and 30% shorter loadup time.
